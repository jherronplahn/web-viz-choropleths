United States Choropleth Maps

Comparing normalized datasets across categories and years


As you explore the maps, some correlations across datasets are more obvious than others. A less obvious correlation would be CO2 Emissions with Chronic Respiratory Disease Mortality Rates per state, with independent T-tests demonstrating statistical significance for all years. Even with normalized data this could be an example of extreme outlier states (like Texas or California) skewing the results.

Plotly provides a simple way to create choropleth maps without the need to provide GeoJSON files for encoding geographic data structures. All maps on this site are responsive embeds using Bootstrap to enable interaction with the live maps.

Maps & Data


The purpose of this site is to demonstrate creating U.S. choropleths maps with Plotly to compare datasets for correlations. The data was cleaned and formatted with Python using the Pandas library. All datasets were normalized prior to mapping.
